# Student Grade Tracker

A Java console application developed as part of the CodeAlpha Java Programming Internship.

## Features
- Add student names and grades
- Calculate average score
- Find highest and lowest scores
- Display summary report
- Uses Object-Oriented Programming (OOP)

## Technologies Used
- Java
- IntelliJ IDEA
- ArrayList
- OOP Concepts
