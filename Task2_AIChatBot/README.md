# AI Chatbot 🤖

A Java-based Artificial Intelligence Chatbot developed as part of the CodeAlpha Java Programming Internship.

## 📌 Project Description

This project simulates an AI chatbot capable of interacting with users through a console-based interface. The chatbot uses rule-based Natural Language Processing (NLP) techniques to understand user queries and provide appropriate responses.

## ✨ Features

- Interactive conversation with users
- Responds to greetings and common questions
- Provides information about Java and OOP concepts
- Displays current system time
- Handles unknown queries gracefully
- Simple and user-friendly console interface
- Exit command to end the conversation

## 🛠 Technologies Used

- Java
- Object-Oriented Programming (OOP)
- String Processing
- Java Time API
- IntelliJ IDEA

## 📚 Concepts Implemented

- Classes and Objects
- Methods
- Conditional Statements
- Loops
- String Handling
- User Input using Scanner
- Rule-Based Chatbot Logic
