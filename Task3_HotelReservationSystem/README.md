# Hotel Reservation System

A Java-based Hotel Reservation System developed as part of the CodeAlpha Java Programming Internship.

## Features
- View available rooms
- Book rooms
- Cancel reservations
- Room categorization
- Menu-driven interface
- Object-Oriented Programming

## Technologies Used

- OOP
- ArrayList
- IntelliJ IDEA

## Room Categories
- Standard
- Deluxe
- Suite
